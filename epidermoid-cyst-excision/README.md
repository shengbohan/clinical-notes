# 皮脂腺囊肿切除术操作指南

面向急诊外科、普外科、皮肤外科、全科和规培医生。  
文献检索与整理日期：2026-06-28。  
本文用于临床学习、科室教学和科普写作，不替代本院制度、上级医师判断、专科会诊意见、病理诊断或患者个体化治疗。

## 一句话总结

临床常说的“皮脂腺囊肿”“粉瘤”多数更准确地说是表皮样囊肿或表皮包涵囊肿。切除的核心不是把包块挖得越深越好，而是在确认适合门诊小手术后，沿皮纹设计足够直视的切口，浅层、逐层、贴囊壁分离，尽量完整取出囊壁和与皮肤相连的开口。尤其在肌肉表面、脂肪层薄、解剖层次不清的部位，宁可适当放大切口、慢慢找层次，也不要一刀切深或盲目向肌肉里追。

近年公开资料反复强调几件事：

- 无症状、典型、稳定的小表皮样囊肿可以观察，不必为了“预防癌变”常规切除。
- 有反复感染、破溃流出角质物、疼痛、摩擦、异味、外观困扰、持续增大或诊断不确定时，可考虑切除或进一步评估。
- 急性红肿热痛、蜂窝织炎或脓肿期不适合勉强完整切除；此时囊壁脆、层次乱，容易破裂、残留和感染扩散。
- 复发的常见原因是囊壁残留、中央开口残留、炎症瘢痕下剥离不完整，或最初诊断并不是普通表皮样囊肿。
- 切口过小会让操作变成盲剥离。困难病例适当扩大切口，通常比在小口里深挖更安全。

## 1. 先判断：这是适合小手术切除的囊肿吗

### 1.1 典型表现

典型表皮样囊肿多位于面颈部、耳后、躯干、背部、臀部和四肢皮下，常表现为缓慢增大的圆形或半球形结节，质地韧或偏硬，边界相对清楚，表面皮肤可见中央黑点或小孔。挤压时可能排出白色或黄白色豆渣样角质物，常有异味。

查体至少写清：

- 部位：具体到左/右、解剖区域，是否靠近眼睑、鼻唇沟、耳前耳后、颈部、乳房、会阴、手足、关节或重要神经血管。
- 大小：长径、短径、高度，最好用厘米记录。
- 深度：皮内、皮下、筋膜浅层，还是疑似筋膜下或肌内。
- 活动度：相对皮肤、皮下和深部组织是否活动。
- 皮肤：中央孔、红肿、破溃、瘢痕、窦道、流脓、色素改变。
- 炎症：压痛、波动感、局部热感、蜂窝织炎、发热。
- 病程：发现多久，是否突然增大，是否曾自行挤压、切开引流或反复感染。

### 1.2 门诊小手术相对适合的情况

可以考虑在局麻下择期切除：

- 浅表皮内或皮下囊肿，边界清楚，活动度可。
- 直径通常小于 3-5 cm，且不是短期快速增大。
- 无明显急性红肿、蜂窝织炎或脓肿。
- 位置远离重要神经血管、腺体、导管、腱鞘和面部危险美容区域。
- 患者能配合局麻操作，出血、感染和基础病风险可控。
- 操作者能在直视下分离完整囊壁，必要时可送病理并安排复诊。

这类小手术仍然要按手术对待：身份核对、部位核对、知情同意、无菌操作、局麻安全剂量、术后随访和病理追踪都不能省。

### 1.3 先影像或转诊的红旗

出现以下情况，不建议在普通门诊或急诊小手术间“试着切一下”：

- 肿块深于皮下，疑似筋膜下、肌内，或随肌肉收缩移动。
- 位于肌肉表面且脂肪层很薄，触诊边界不清，术前不能确定层次。
- 直径大于或等于 5 cm，短期快速增大，质地硬，固定，或疼痛进行性加重。
- 无典型中央孔，形态不规则，皮肤溃疡、出血、色素异常或坏死。
- 既往反复切开引流、复发、瘢痕明显，包膜可能与周围组织粘连。
- 伴持续发热、广泛蜂窝织炎、免疫抑制、糖尿病控制差或全身感染风险。
- 位于眼睑、内外眦、鼻翼、唇部、耳前腮腺区、颈后三角、锁骨上、腋窝、腹股沟、腕掌侧、手指、足底、乳头乳晕、外阴阴囊等特殊部位。
- 儿童中线、鼻根、眉外侧、骶尾部或头颈部先天性样囊肿，需警惕皮样囊肿、窦道或深部交通。

实用阈值：

- 摸不清层次、不能确认在皮下者，优先做超声。
- 深部、复发、较大、固定或影像不典型者，不要按普通粉瘤切；按软组织肿物流程处理。
- 一旦怀疑肿瘤性病变，切口设计和活检路径会影响后续治疗，不要随意横切、碎取或盲剥。

## 2. 术前准备

### 2.1 病史和风险评估

术前询问：

- 发现多久，是否持续变大或突然变大。
- 有无疼痛、红肿、流脓、破溃、异味和发热。
- 是否反复挤压、针挑、切开引流或外用腐蚀性药物。
- 既往是否切过，病理结果和复发时间。
- 抗凝/抗血小板药、糖皮质激素、免疫抑制剂、糖尿病、肝肾病、出血史。
- 局麻药、碘伏、氯己定、胶布、抗生素过敏史。
- 瘢痕体质或既往瘢痕增生。
- 妊娠、哺乳、起搏器、不能配合体位等特殊情况。

典型小囊肿通常不需要常规血常规、凝血和影像全套。高龄、多病、抗凝、糖尿病、感染、较大肿物、特殊部位或需要更大范围操作时，应按本院术前流程补齐。

### 2.2 知情同意要说清

建议把以下风险讲明并记录：

- 出血、皮下瘀斑、血肿、浆液肿。
- 感染、切口裂开、延迟愈合，炎症期切除风险更高。
- 囊肿破裂、角质物污染创面、囊壁残留、复发。
- 疼痛、麻木、感觉异常，少数可持续存在。
- 神经、血管、肌肉、腱鞘、腺体或导管损伤，具体风险取决于部位。
- 瘢痕、色素沉着、凹陷、皮肤皱褶或外观不满意。
- 病理结果可能不是普通表皮样囊肿，若提示其他病变，可能需要进一步影像、会诊和治疗。

### 2.3 器械和材料

常用准备：

- 记号笔、尺子、拍照设备。
- 无菌铺巾、手套、纱布、棉签、冲洗液。
- 皮肤消毒剂：碘伏或氯己定，按本院规范和禁忌选择。
- 局麻药：常用 1% 或 2% 利多卡因，可按部位和患者情况考虑是否加肾上腺素。
- 11 号或 15 号刀片、眼科剪或小组织剪、蚊式钳、止血钳、有齿/无齿镊。
- 小拉钩或皮肤钩，便于暴露浅层囊壁。
- 电凝或其他止血工具，如小手术间具备。
- 可吸收缝线用于关闭死腔或皮下减张，尼龙线或其他皮肤缝线用于皮肤关闭。
- 无菌敷料、弹力绷带或加压材料。
- 病理标本瓶和申请单。

局麻提醒：

- 先标记再打麻药，避免局部肿胀后边界和中央孔消失。
- 麻药尽量打在囊肿周围和切口旁，不要把针反复穿进囊腔。
- 每次注射前回抽，避免血管内注射。
- 严格计算局麻药最大剂量；儿童、低体重、肝病、妊娠或多处操作时更要保守。
- 手指、足趾、鼻尖、耳廓、阴茎等血供特殊部位是否使用含肾上腺素局麻，应遵循本院规范和操作者经验。

## 3. 标准开放切除步骤

### 3.1 体位、标记和切口设计

1. 让患者取能长时间保持的体位，充分暴露肿物，避免术中移动。
2. 触诊并画出囊肿边界、中央孔、皮纹方向和计划切口。
3. 优先沿皮纹或张力线设计切口；面部和关节附近需兼顾瘢痕方向和张力。
4. 有中央孔、反复流出角质物或皮肤粘连者，可设计小梭形切口，把中央孔和粘连皮肤一并切除。
5. 切口不必过大，但必须足够直视分离。囊壁薄、炎症后粘连、脂肪层薄或靠近肌肉表面时，切口可以适当放大。
6. 不为了追求“小疤痕”把操作变成盲剥离。看不见囊壁和深面层次，就不是合适的切口大小。

### 3.2 消毒、铺巾和局麻

1. 完成身份、部位和操作核对。
2. 消毒范围要足够大，尽量保留关键标记。
3. 铺无菌巾后进行局麻。先皮内，再沿切口和囊肿周围做场阻滞。
4. 等待麻药起效后，用镊夹或针刺确认痛觉阻滞。
5. 如果局麻后边界摸不清，先重新定位，不要凭印象深挖。

### 3.3 切开和显露囊壁

1. 沿设计线切开皮肤，浅层进入，刀尖方向要可控。
2. 第一刀只解决皮肤和极浅皮下，不要一刀扎到深部。
3. 到达囊壁附近时，通常可见灰白、乳白或珍珠样薄壁结构，内容物可能透出黄白色。
4. 囊壁很薄，尤其在炎症后、肌肉表面或脂肪组织少的部位，不要用刀大幅度向深部切。
5. 用小剪刀、蚊式钳或棉签在直视下找囊壁外侧间隙，先分清“囊壁”和“正常皮下/筋膜/肌肉”。
6. 若越切越深仍找不到囊壁，或看到筋膜、肌肉纤维而囊肿层次不清，应停下重新评估；不要把深部组织当成囊壁继续削。

### 3.4 沿囊壁完整分离

1. 用皮钩、小拉钩或镊子轻柔牵开皮缘，不要用力夹碎囊壁。
2. 围绕囊壁外侧做钝性分离，遇到粘连带再在直视下锐性剪断。
3. 牵拉要轻，囊壁要向上提，剪刀尖朝向可见的安全间隙。
4. 分离顺序可先处理浅面和两侧，再处理深面；深面是最容易切过深的地方。
5. 肌肉表面、前臂小腿、肩胛区、胸壁、面颈部等脂肪层薄的部位，深面分离尤其要浅。看到肌膜或肌肉纹理后，不要继续向下挖。
6. 包膜完整优先。能整囊取出时，不主动切开囊肿，不反复挤压内容物。
7. 如果囊壁破裂，先用纱布保护创缘，清除角质物，冲洗后在直视下寻找剩余囊壁。必要时扩大切口，把残余囊壁和中央孔清除干净。
8. 不要把“包膜切干净”理解成切除一圈正常肌肉或筋膜。真正需要清除的是囊壁、中央孔和明确粘连的病变组织。
9. 囊肿与皮肤中央孔紧密相连时，要把这段皮肤小梭形切除；只掏出内容物而留下开口和囊壁，复发风险高。

### 3.5 囊肿取出后的检查

1. 检查标本是否完整：是否有完整囊壁、中央孔、破口和残缺处。
2. 观察创腔四周，有无白色囊壁残片、角质碎屑或窦道。
3. 对可见残留囊壁，用小剪刀或镊子在直视下清除。
4. 用生理盐水冲洗创腔，清除角质物和血凝块。
5. 炎症后腔隙容易渗血和残留碎屑，止血前先把视野清干净。

### 3.6 止血、关闭和包扎

1. 松开牵引后观察渗血点，逐一电凝、压迫或缝扎。
2. 小囊肿腔隙通常不大，可直接皮肤缝合；较大、背部、肩部或死腔明显者，可用可吸收线关闭皮下死腔。
3. 不要把皮肤强行缝得很紧来消灭死腔；这会增加皮缘缺血、裂开和瘢痕。
4. 污染明显、急性感染后切开引流或脓腔病例，不按清洁切除强行一期紧密关闭，应按感染创面处理。
5. 皮肤按部位选择间断缝合、皮内缝合、胶水或胶条。张力大的部位优先可靠减张。
6. 无菌敷料覆盖，必要时加压包扎，尤其背部、肩部、胸壁和切除腔较大的病例。

### 3.7 标本处理

建议切除标本送病理，尤其是以下情况：

- 术前诊断不确定。
- 直径接近或超过 5 cm。
- 近期快速增大、复发、破溃、出血或形态不典型。
- 术中囊壁颜色、质地、出血、粘连程度不典型。
- 患者因为“排除恶性”而要求切除。

病理申请单写清部位、大小、病程、是否复发、是否曾感染或切开引流、临床诊断和术中所见。标本如果破裂或分块送检，也要说明。

## 4. 小切口、打孔和切开引流：边界在哪里

### 4.1 小切口或打孔取囊

小切口或打孔技术适合非常浅表、较小、未感染、囊壁完整、远离重要结构、中央孔明确且操作者熟练的病例。它的优势是瘢痕小，但对层次判断和囊壁完整性要求更高。

不建议作为困难病例的默认选择：

- 近期红肿、破裂、流脓或反复挤压。
- 既往切开引流后复发，瘢痕粘连明显。
- 脂肪层薄，囊肿贴近肌肉、筋膜、腱鞘或重要结构。
- 囊肿较大、张力高或囊壁很薄。
- 术中一拉就破、看不清深面或出血影响视野。

原则：小切口不是盲切口。看不清、拉不出、疼痛明显、出血多或囊壁反复破裂时，应扩大切口、改为开放切除或停止。

### 4.2 单纯切开挤出内容物

单纯切开、挤出豆渣样内容物只能暂时减压，不能等同于根治切除。只要囊壁和中央开口残留，就有复发可能。

适用场景主要是急性脓肿或明显波动感，需要先引流减压，而不是追求一期完整切除。炎症消退、瘢痕软化后，再评估是否择期完整切除。

### 4.3 急性炎症或感染期

急性红肿热痛时，囊壁常脆且与周围组织粘连，完整切除困难。处理思路：

- 只有轻度无脓性炎症：可先消炎、止痛、局部护理，待炎症消退后择期切除。
- 有波动感、脓液或明显脓肿：按脓肿切开引流处理，必要时留取培养。
- 有蜂窝织炎、发热、糖尿病、免疫抑制、面部危险区域或感染范围较大：按本院流程评估抗菌治疗和专科会诊。
- 引流后不要忘记复诊。脓肿好了不等于囊壁消失，复发者需择期切除囊壁。

## 5. 术后处理

### 5.1 当场交代

术后至少交代：

- 敷料按本院要求保留，通常 24-48 小时内避免浸水。
- 当日避免剧烈活动、牵拉切口、泡澡、游泳和饮酒。
- 疼痛多可用对乙酰氨基酚或 NSAIDs；有出血风险、肾病、胃病或抗凝患者需个体化。
- 少量渗血和瘀斑可以观察；持续出血、敷料快速浸透或局部肿胀快速变大要立即返院。
- 病理结果必须追踪，不要把“看起来像粉瘤”当最终诊断。

### 5.2 抗生素

未感染的清洁切除通常不需要常规预防性抗生素。是否用药取决于：

- 是否存在急性感染、蜂窝织炎或脓肿。
- 患者是否糖尿病控制差、免疫抑制、人工瓣膜或其他高风险状态。
- 部位是否特殊，污染是否明显，是否有大量角质物污染创面。
- 本院围手术期抗菌药物制度和当地耐药情况。

### 5.3 复诊和拆线

可参考：

- 术后 2-7 天复查切口，尤其较大囊肿、曾感染、死腔明显、糖尿病、抗凝或免疫抑制患者。
- 面部 5-7 天拆线。
- 头皮、躯干和上肢多为 7-14 天。
- 下肢、背部张力大区域或关节附近可延至 14-21 天。

具体时间按切口张力、愈合情况和本院规范调整。

### 5.4 立即返院信号

- 切口红肿热痛进行性加重、流脓、发热。
- 出血不止、局部包块迅速变大、疼痛明显加重。
- 远端麻木、乏力、发凉、颜色改变。
- 切口裂开、皮缘发黑或明显渗液。
- 病理提示非表皮样囊肿、恶性或诊断不明确。

## 6. 常见并发症和处理思路

### 6.1 囊壁破裂和内容物污染

囊壁破裂不罕见，尤其炎症后、反复挤压或囊壁很薄的病例。处理重点不是慌着深挖，而是：

- 保护创缘，清除角质物。
- 适当扩大切口，恢复直视。
- 找到残余囊壁，逐片完整清除。
- 充分冲洗，必要时根据污染和感染情况决定是否一期关闭。

### 6.2 复发

复发常见原因是囊壁、中央孔或窦道残留，也可能是急性期只做了切开引流。复发囊肿往往有瘢痕粘连，二次手术前建议先评估炎症是否完全消退，必要时做超声确认边界和深度。

### 6.3 感染

未感染囊肿切除后出现局部红肿热痛、脓性渗出或发热，应按感染切口处理：拆除部分缝线引流、培养、冲洗、换药和抗菌治疗需按范围及全身情况决定。

### 6.4 血肿或浆液肿

小血肿可压迫、制动和随访。快速增大、疼痛明显、影响皮瓣血供或压迫神经血管时，应尽快打开切口清除血凝块、重新止血。较大腔隙或背部病变要重视关闭死腔和加压包扎。

### 6.5 麻木、瘢痕和外观问题

面部、耳周、颈部、手足和关节附近，术前要说明神经刺激、麻木和瘢痕风险。瘢痕体质、张力大、感染或反复切开引流者，术后瘢痕增生风险更高。

## 7. 操作中的停止条件

遇到以下情况，及时停手比“再试一下”更安全：

- 局麻仍明显疼痛，患者无法配合。
- 囊壁找不到，越分越深，已经接近筋膜或肌肉。
- 肿物固定于筋膜、肌肉、腺体、腱鞘或神经血管束。
- 出血多，无法在当前切口内看清。
- 囊壁反复破裂，角质物污染明显，创面层次混乱。
- 质地硬、灰白、鱼肉样、坏死、血供丰富或明显不像普通囊肿。
- 出现远端麻木、运动异常、皮肤缺血或患者明显不适。
- 术中发现实际大小和深度明显超过术前判断。

停手后可以压迫止血、临时关闭或开放引流、补充超声或转上级。小手术最怕的不是没切完，而是不知道什么时候该停。

## 8. 病历模板

可以改成本院电子病历短语：

```text
患者因[部位]皮下肿物[时长]就诊。肿物增长情况：[缓慢/近期增大/不详]，症状：[无痛/疼痛/反复红肿/破溃流出豆渣样物/异味/影响外观/反复摩擦]。既往类似肿物或切开引流/切除史：[有/无]，抗凝抗血小板药物：[有/无]，过敏史：[ ]。

查体：[左/右][部位]可及皮内/皮下肿物，约[ ] cm x [ ] cm，质地[韧/硬/软]，边界[清/欠清]，活动度[好/差]，与皮肤[不粘连/粘连]，中央孔[有/无]，与深部[活动/固定]，表面皮肤[正常/红肿/破溃/瘢痕/流脓]，波动感[有/无]，局部压痛[有/无]，远端感觉运动血运[正常/异常]。

评估：考虑[表皮样囊肿/皮脂腺囊肿/感染性囊肿/其他软组织肿物]。已向患者说明观察、影像评估、切除、切开引流及转诊等选择，并说明出血、血肿、感染、切口裂开、瘢痕、麻木、神经血管损伤、囊壁破裂、包膜残留、复发、病理结果不符预期及需进一步治疗等风险。患者/家属表示理解并同意手术。

操作：核对身份、部位和术式后，常规消毒铺巾。术前标记肿物边界、中央孔和切口。予[局麻药名称、浓度、剂量]局部浸润/场阻滞，麻醉满意后沿皮纹/梭形切开约[ ] cm，浅层逐层分离至囊壁，于直视下沿囊壁钝性及锐性分离，完整/分块取出囊性肿物约[ ] cm x [ ] cm，内容物[未外溢/可见黄白色角质物外溢，已清除并冲洗]。检查创腔未见明显囊壁残留。彻底止血，冲洗，关闭死腔[有/无]，皮肤以[缝线/胶水/胶条]关闭，加压包扎。标本已送病理。

术后：患者一般情况可，远端感觉运动血运[正常/同术前]。交代换药、保持干燥、避免牵拉和剧烈活动、复诊拆线、病理追踪及出血、感染、麻木、切口裂开等返院指征。
```

急性感染或脓肿期记录可改为：

```text
操作：局麻满意后，于波动最明显处/原破溃处切开约[ ] cm，见脓液及黄白色角质样物排出约[ ] mL，留取培养[是/否]。予钝性分离脓腔，生理盐水反复冲洗，置入引流条/纱条，无菌敷料包扎。已告知本次为切开引流，不能保证囊壁完整去除，炎症消退后需复诊评估是否择期切除。
```

## 9. 给自己的复盘问题

- 我有没有确认这是浅表、典型、适合小手术处理的囊肿？
- 我有没有记录大小、深度、中央孔、炎症状态、活动度和神经血管状态？
- 我有没有先标记边界和中央孔，再局麻？
- 我的切口是否足够直视分离，而不是为了小疤痕变成盲剥离？
- 在脂肪层薄或肌肉表面的部位，我有没有避免一刀切深、避免向肌肉里追？
- 我有没有尽量完整取出囊壁和中央孔，确认创腔无明显残留？
- 如果囊壁破裂，我有没有清除角质物、冲洗并寻找残余囊壁？
- 我有没有把标本送病理，并安排病理结果追踪？
- 如果术中发现不像普通囊肿，我有没有及时停止并转上级评估？
