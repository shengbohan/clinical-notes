# 急诊外科临床笔记 Clinical Notes

这是一个面向急诊外科临床工作的个人知识库，用于整理常见急诊外科疾病、创伤处置、床旁操作、复盘模板和患者科普内容。

> 说明：本文档用于临床学习、科室教学和科普写作，不替代本院诊疗规范、上级医师判断、专科会诊意见或患者个体化诊疗。
